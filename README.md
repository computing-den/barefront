# Barefront

Bare minimum create React app.
This project is still a work in progress.

# Development
npm run dev
npm run chmod
npm link

Usage:
npx barefront create my-project --linkToBarefront

See the generated project's README.md for its usage.

# Production
npm run build
npm run chmod

Usage:
npx barefront create my-project

See the generated project's README.md for its usage.
